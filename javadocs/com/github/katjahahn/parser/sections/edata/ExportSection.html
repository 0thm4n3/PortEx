ExportSection

class ExportSection extends SpecialSection

Represents the export section of a PE file and provides access to lists of it's inner structures (export address table, ordinal table, name pointer table, data directory table) as well as access to a list of export entries fetched from these structures.

The export section instance should be created with the SectionLoader
